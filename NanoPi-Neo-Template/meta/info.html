This project template is the basis of an expansion board for the NanoPi NEO series of ARM single board computers.

This base project includes a PCB edge defined as the same size as the NanoPi NEO PCB with the connectors placed correctly to align the two boards. All IO present on the NEO board is connected to the project through the 0.1" expansion headers.
